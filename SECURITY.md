# Security Policy for ExecuNova AI

We take security seriously. Please follow this policy if you discover a vulnerability.

## Reporting a Vulnerability

1. **Email us privately** at: `ankitkumarforall@gmail.com`  
2. **Do not post publicly** (e.g., GitHub Issues)  
3. Include the following information:
   - Steps to reproduce the issue  
   - Potential impact  
   - Screenshots or logs (if applicable)  
   - Environment details (OS, browser, Node version)

## Response

- We will respond within **48 hours**.  
- Confirm receipt of your report and give status updates.  
- Fixes will be prioritized based on severity.  

## Responsible Disclosure

- Security researchers reporting responsibly may be credited in the README.  
- Exploiting vulnerabilities without reporting is prohibited.
