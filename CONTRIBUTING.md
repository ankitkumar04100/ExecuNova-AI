# Contributing to ExecuNova AI

First of all, thank you for considering contributing! Your help improves the platform and helps others avoid missed deadlines.

## How to Contribute

1. **Fork the Repository**  
   Click "Fork" in the top-right corner to create your copy.

2. **Clone Your Fork**  
   ```bash
   git clone https://github.com/your-username/ExecuNova-AI.git
   cd ExecuNova-AI
   ```
   
3. **Create a Feature Branch**
   ```bash
   git checkout -b feature/<feature-name>
   ```
   Use descriptive names like feature/risk-dashboard or feature/ai-optimizer.

4. **Make Your Changes**

- Follow existing code style
- Comment complex logic
- Keep commits small and descriptive

5. **Run Tests**
   Ensure your changes don’t break anything:
  ```bash
  npm test --prefix frontend
  npm test --prefix backend
  ```

6. **Push & Create Pull Request**
   ```bash
   git push origin feature/<feature-name>
   ```
   Open a pull request against dev branch.

## Guidelines

- Follow the Code of Conduct
- Make sure your code is modular, readable, and tested
- Include screenshots or demo GIFs for UI changes

Contributions of all sizes are welcome — even fixing typos in documentation!
